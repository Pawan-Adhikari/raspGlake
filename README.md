# Original Firmware for Raspberry Pi Pico.

### Sensors Used:
- MPU 6050 (IMU)
- BMP280
- Analog Humidity sensor
- k-Type Thermocouple Temperature Sensor

#### This Raspberry Pi pico is connected to a Raspberry Pi 4 and thus, programming, debugging, and serial transmission is done live from that device.

## Successful deployment at Gangapurna Lake, Manang!
